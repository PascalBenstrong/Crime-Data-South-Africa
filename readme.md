## Running the project

### install the required packages

- linux

```shell
python3 -m pip install -r requirements.txt
```

- windows

```shell
python -m pip install -r requirements.txt
```

or

```shell
py -m pip install -r requirements.txt
```

### run the app

- linux

```shell
python3 main.py
```

- windows

```shell
python main.py
```

or

```shell
py main.py
```
